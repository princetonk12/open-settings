Getting Started

Article written by Manu Anish

This is the home page for the documentation. This page will contain some general information about the project and some FAQ.


How does it work?

To bypass the local network firewall the project connects to a live server around the world and requests the data from there instead. This uses the NPM package corrosion to allow for requests to be made within the webpage and displayed to the user.

What hosting platform do you use?

To host the website, I use the free Cloud Application Platform Heroku. This allows me to host the application for free 24/7.
