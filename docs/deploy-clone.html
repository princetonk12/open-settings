Depolying a Clone

Article written by Manu Anish


In this page we will discuss how to create your own clone of the application and run it on a free server 24/7.

Prerequisites

To follow along with this tuotorial, you will need to have:

  • A Github Account
  • A Heroku Account


Creating a Heroku App

Once we have created a Heroku account, we can navigate to the Dashboard and create a new personal app.

The name of this app will be the domain used to connect to the application: [app-name].herokuapp.com. We can change the name of the app in the Settings tab.

Cloning the Application

Forking the Repository

Once we have logged in into our Heroku Account, we can clone the repository by clicking on the fork button. Once you do this, you have a copy of the code on your account.

Connecting Github to Heroku

We can now connect the application to Heroku. In the Deploy section, we have the option to connect our Github account to Heroku. Once we finish we can serach open-settings to connect the repository.


Deploying the Application

We can now deploy the master branch by clicking on the Deploy Branch button; the app is now live. To connect we can use the URL: 

https://[app-name].herokuapp.com
